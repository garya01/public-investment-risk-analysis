# Public Investment Risk Analysis – Dominican Republic

This project provides a complete financial and risk analysis framework for public investment projects using real-world data from MEPyD.

## 🔍 Overview

The analysis evaluates project performance through:

- Net Present Value (NPV)
- Sector-level investment efficiency
- Temporal investment trends
- Geographic distribution of public spending
- Risk assessment using Monte Carlo simulation

## ⚙️ Methodology

- Data cleaning and transformation (R / tidyverse)
- Discounted cash flow modeling (NPV)
- Statistical analysis and visualization
- Geospatial analysis using shapefiles
- Risk simulation (10,000 iterations)

## 📊 Key Outputs

- NPV distribution across projects
- Top-performing sectors by investment return
- Investment trends over time
- Geographic heatmap of investment
- Monte Carlo simulation (risk distribution)
- Cumulative probability analysis (CDF)

## 🧰 Tools & Technologies

- R (tidyverse, ggplot2, sf)
- RMarkdown
- Data visualization
- Monte Carlo simulation

## 💡 Business Value

This model can be used for:

- Public investment evaluation
- Risk analysis in infrastructure projects
- Financial feasibility studies
- Policy decision support
- Economic research
